# EXIF Data Viewer

Расширение для браузера Google Chrome, позволяющее просматривать метаданные EXIF изображений прямо в браузере.

[English](README.md) | Русский

## Возможности

- **Простота использования**: Просто кликните правой кнопкой мыши по любому изображению и выберите "View EXIF Data"
- **Базовый и расширенный просмотр**: По умолчанию отображается основная информация EXIF, с возможностью расширения для просмотра детальных метаданных
- **Конфиденциальность**: Вся обработка происходит локально в вашем браузере. Данные не собираются и не передаются
- **Работает везде**: Совместимо со всеми веб-сайтами и локальными файлами
- **Легковесное**: Минимальное влияние на производительность

## Установка

### Из Chrome Web Store
*(Скоро)*

### Ручная установка

1. Скачайте или клонируйте этот репозиторий
2. Откройте Chrome и перейдите на `chrome://extensions/`
3. Включите "Режим разработчика" в правом верхнем углу
4. Нажмите "Загрузить распакованное расширение" и выберите директорию расширения
5. Расширение установлено и готово к использованию

## Использование

1. Перейдите на любую веб-страницу с изображениями
2. Кликните правой кнопкой мыши по изображению, которое хотите проверить
3. Выберите "View EXIF Data" из контекстного меню
4. Всплывающее окно отобразит базовую информацию EXIF
5. Нажмите "Show more" для просмотра дополнительных детальных метаданных

## Что такое EXIF?

EXIF (Exchangeable Image File Format) — это стандарт, определяющий форматы для изображений, звука и вспомогательных тегов, используемых цифровыми камерами, сканерами и другими системами обработки файлов изображений и звука. Данные EXIF могут включать:

- Настройки камеры (ISO, диафрагма, выдержка)
- Дата и время съемки
- GPS координаты (если были включены службы определения местоположения)
- Производитель и модель камеры
- Программное обеспечение, использованное для редактирования изображения
- И многое другое

## Конфиденциальность

Это расширение уважает вашу конфиденциальность:
- Вся обработка данных EXIF происходит локально в вашем браузере
- Данные не отправляются на внешние серверы
- Личная информация не собирается
- Отсутствует отслеживание и аналитика

## Технические детали

- **Версия манифеста**: 3
- **Библиотека**: Использует [exifr](https://mutiny.cz/exifr/) для парсинга данных EXIF
- **Разрешения**: 
  - `activeTab`: Для доступа к текущей вкладке
  - `contextMenus`: Для добавления опции в контекстное меню
  - `scripting`: Для внедрения контентных скриптов

## Разработка

Это расширение построено с использованием:
