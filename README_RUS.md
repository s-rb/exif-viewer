## Возможности

- **Простота использования**: Просто кликните правой кнопкой мыши по любому изображению и выберите "View EXIF Data"
- **Базовый и расширенный просмотр**: По умолчанию отображается основная информация EXIF, с возможностью расширения для просмотра детальных метаданных
- **Конфиденциальность**: Вся обработка происходит локально в вашем браузере. Данные не собираются и не передаются
- **Работает на большинстве сайтов**: Совместимо с подавляющим большинством веб-сайтов
- **Легковесное**: Минимальное влияние на производительность

> **Важно**: На некоторых сайтах изображения показываются как предпросмотр или сильно оптимизированные копии (например, уменьшенные превью, конвертированные в WebP/AVIF, сжатые CDN‑версии). В таких файлах EXIF может быть удалён или присутствовать только в исходном оригинале. Подробнее см. разделы **«Ограничения»** и **«Работа с превью и оптимизированными картинками»** ниже.

## Обзор

EXIF Data Viewer — это расширение для Chrome, которое позволяет смотреть EXIF‑метаданные фотографий прямо в браузере по клику правой кнопкой мыши. Узнайте, на какую камеру и с какими настройками сделан кадр, где и когда он снят — без скачивания файлов и без перехода на сторонние сайты.

### Зачем использовать EXIF Data Viewer?

- **Один клик по картинке**: Кликните правой кнопкой мыши по изображению и выберите «View EXIF Data» — всплывающее окно с данными появится рядом с фото.
- **Удобно фотографам и авторам контента**: Камера, объектив, диафрагма, выдержка, ISO, фокусное расстояние, дата съёмки, координаты и другие параметры сгруппированы и отформатированы так, чтобы их было удобно читать.
- **Идеально для соцсетей и блогов**: Быстро создавайте красиво оформленный блок с основными настройками съёмки (например, для Instagram) и копируйте его в буфер обмена.
- **Максимум приватности**: Расширение не отправляет ваши изображения и метаданные на сервер. Весь разбор EXIF происходит локально в вашем браузере.
- **Работает на большинстве сайтов**: Поддерживаются обычные `<img>`‑теги и локальные файлы, открытые в браузере.

### Основные возможности

- Базовый и расширенный режимы просмотра EXIF в компактном окошке
- Показ камеры, объектива и ключевых параметров съёмки (диафрагма, выдержка, ISO, фокусное расстояние)
- Отображение даты и времени съёмки в удобном человекочитаемом формате
- GPS‑координаты с быстрым переходом в Google Maps (если данные есть в файле)
- Готовый форматированный блок с настройками для Instagram и других площадок
- Расширенный технический раздел для продвинутых пользователей
- Поддержка множества вариаций EXIF от разных производителей камер

### Ограничения

Как и любой EXIF‑просмотрщик, расширение зависит от того, **какой именно файл отдает сайт**:

- **Если в файле нет EXIF — показывать нечего**: Многие сайты и сервисы по умолчанию удаляют метаданные (по соображениям приватности или для уменьшения размера файлов). В таком случае EXIF физически отсутствует, и восстановить его невозможно.
- **Предпросмотр / оптимизированные изображения**: Часто вместо оригинала страница загружает уменьшенное превью или перекодированную версию (WebP/AVIF, сильное сжатие через CDN). В этих копиях EXIF обычно вырезан, даже если в исходном фото он есть.
- **Необычные способы отображения картинок**: Изображения, нарисованные на `<canvas>`, использованные только как CSS‑фоны или встроенные в сложные JS‑виджеты, могут быть недоступны как обычные URL‑файлы. В таких случаях расширение технически не может достать EXIF.

В этих ситуациях расширение покажет сообщение вроде «EXIF‑данные не найдены» — это нормальное поведение и означает, что страница просто **не предоставляет** EXIF для данного изображения.

### Работа с превью и оптимизированными картинками

Если вы кликнули по миниатюре/превью и EXIF‑данные не отобразились, попробуйте следующее:

1. **Откройте оригинальное изображение**  
   - Найдите кнопку или ссылку вида «Открыть оригинал», «Просмотр в полном размере», «Download», иконку увеличения/лупы и т.п.  
   - На многих галереях и блогах нужно один‑два раза кликнуть по картинке, чтобы открыть крупную или оригинальную версию.

2. **Используйте расширение уже на оригинале**  
   - Когда изображение открылось в отдельной вкладке или в полноэкранном просмотрщике, кликните правой кнопкой **именно по этой картинке**.  
   - Снова выберите в контекстном меню **«View EXIF Data»**.

3. **Если EXIF всё равно не появился**  
   - Скорее всего, сайт полностью удаляет метаданные или отдаёт только перекодированные копии без EXIF.  
   - В таком случае ни одно расширение не сможет показать EXIF напрямую с сайта — данных просто нет.

4. **Альтернатива: просмотреть EXIF из локального файла**  
   - Скачайте фотографию на компьютер.  
   - Перетащите файл в новое окно/вкладку браузера (или откройте через меню `Файл → Открыть файл` в браузере).  
   - Кликните правой кнопкой по изображению в этой вкладке и выберите **«View EXIF Data»** — расширение прочитает EXIF напрямую из локального файла, если он там присутствует.

## Использование

1. Перейдите на любую веб-страницу с изображениями
2. Кликните правой кнопкой мыши по изображению, которое хотите проверить
3. Выберите "View EXIF Data" из контекстного меню
4. Всплывающее окно отобразит базовую информацию EXIF
5. Нажмите "Show more" для просмотра дополнительных детальных метаданных

## Что такое EXIF?

EXIF (Exchangeable Image File Format) — это стандарт, определяющий форматы для изображений, звука и вспомогательных тегов, используемых цифровыми камерами, сканерами и другими системами обработки файлов изображений и звука. Данные EXIF могут включать:

- Настройки камеры (ISO, диафрагма, выдержка)
- Дата и время съемки
- GPS координаты (если были включены службы определения местоположения)
- Производитель и модель камеры
- Программное обеспечение, использованное для редактирования изображения
- И многое другое
